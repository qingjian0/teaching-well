# 教案活井

> "活水润心田，智慧泽教育"

## 项目简介

教案活井是一个基于**2022年版义务教育课程标准**设计的智能教学辅助工具，为乡村教师提供高质量教案生成、分层教学设计、学科实践活动规划等核心功能，帮助教师高效完成教学准备工作。

### 品牌故事

"教案活井"源自《易经》井卦智慧：
- **井之德**：取之不尽，用之不竭
- **井之象**：活水滋养，万物生长
- **井之愿**：为乡村教育注入源头活水

## 核心价值

| 维度 | 说明 |
|------|------|
| **用户** | 乡村小学教师 |
| **场景** | 备课、教学设计 |
| **痛点** | 资源匮乏、备课负担重、质量参差 |
| **方案** | 智能生成、专业标准、社区共享 |

## 核心功能

### 1. 教案智能生成（核心功能）
- ✅ **核心素养导向**：完全对齐2022新课标
- ✅ **12种教案模板**：基础/互动/翻转课堂/项目式学习/情境教学/分层教学/游戏化/探究式/跨学科融合/实验教学
- ✅ **12个学科支持**：语文/数学/英语/科学/道德与法治/历史/地理/音乐/美术/体育/信息技术
- ✅ **多种教材版本**：人教版/部编版/北师大版/沪教版/通用版

### 2. 增值配套功能
- 📝 **配套练习题生成**：选择题+填空题+简答题，三级难度可调
- 📊 **分层教学设计**：基础层/标准层/拓展层三层设计
- 🔍 **学情分析报告**：先备知识+常见误区+学习路径

### 3. 教学质量保障
- ✅ **学业质量评价标准**：四级评定（优秀/良好/中等/待提高）
- ✅ **学科实践活动设计**：体现"做中学"新课标要求
- ✅ **德育与价值观融合**：思政元素+传统文化渗透

## 技术特点

| 特点 | 说明 |

## 使用指南

### 快速开始
1. 访问在线地址：https://qingjian0.github.io/teaching-well/
2. 点击"教案设计"进入核心功能
3. 选择学科、年级、教材版本
4. 输入课题名称
5. 点击"生成教案"
6. 根据需要生成配套练习、分层设计、学情分析

### 界面预览
- **教案生成器**：支持12种模板，实时预览
- **练习题生成器**：一键生成，选择/填空/简答
- **分层教学设计**：三层差异化教学任务
