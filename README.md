# 教案活井

## 项目简介
教案活井是一个专注于教案设计的轻量级智能教学工具，以"活水润心田，智慧泽教育"为理念，提供高质量的教案生成、课件制作和教学辅助功能，帮助教师高效完成教学准备工作。

## 品牌定位
"教案活井"寓意为教师提供源源不断的教学智慧和资源，如同活井般滋养教育事业，为乡村教育注入新的活力。

## 核心功能
- **教案设计**：智能化教案生成，支持多种教学场景
- **课件制作**：一键生成精美课件，省时省力
- **重难点解析**：深度分析教学内容，提升教学质量
- **作业批改**：智能辅助批改，个性化反馈建议

## 技术栈
- **前端**：HTML5 + CSS3 + JavaScript
- **AI能力**：TRAE SOLO SDK

## 如何使用
1. 打开 `index.html` 文件进入应用首页
2. 点击导航菜单选择需要的功能
3. 按照提示输入相关信息
4. 系统将生成相应的教学资源

## 设计理念
- **轻量级**：无需复杂配置，开箱即用
- **专注**：聚焦教案设计核心功能
- **美观**：简洁优雅的视觉设计
- **高效**：快速响应，流畅体验

## 开发说明
- 本项目使用纯前端技术栈，无需后端服务
- 集成了TRAE SOLO SDK，用于提供AI能力
- 支持离线使用核心功能
