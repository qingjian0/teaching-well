# 乡村教师教学助手系统使用指南

## 1. 系统简介

乡村教师教学助手是一个面向乡村教师群体的智能化教学工具，旨在为乡村教师提供持续、稳定的教学支持，减轻工作负担，提升教学质量。

### 1.1 核心功能

- **课件/教案生成**：根据教学目标和内容自动生成课件和教案，支持不同学科和年级的内容适配
- **重难点解析**：自动识别教学内容中的重难点，提供详细的解析和课堂讲解脚本
- **作业批改辅助**：提供作业批改的参考标准，生成个性化的学生反馈建议
- **教学资源管理**：构建本地教学资源库，支持资源的分类、搜索和复用

## 2. 系统安装与访问

### 2.1 系统要求

- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- 网络连接（首次加载需要联网，部分功能支持离线使用）

### 2.2 访问方式

1. **本地访问**：直接打开项目目录中的 `index.html` 文件
2. **服务器访问**：将项目部署到Web服务器后，通过服务器地址访问

## 3. 功能使用指南

### 3.1 课件生成

1. **进入页面**：点击导航菜单中的「课件生成」或「教案设计」
2. **填写信息**：
   - 选择学科（语文、数学、英语、科学、其他）
   - 选择年级（一年级至六年级）
   - 输入教学主题
   - 输入教学内容要求（可选）
   - 选择课件模板（基础模板、互动模板、视觉模板、测验模板）
3. **生成课件**：点击「生成课件」按钮
4. **查看结果**：生成完成后，系统会显示课件内容
5. **操作选项**：
   - 预览课件：查看课件效果
   - 导出课件：下载课件文件
   - 重新生成：根据相同参数重新生成课件

### 3.2 重难点解析

1. **进入页面**：点击导航菜单中的「重难点解析」
2. **填写信息**：
   - 选择学科
   - 选择年级
   - 输入教学主题
   - 输入教学内容
3. **分析重难点**：点击「分析重难点」按钮
4. **查看结果**：
   - 重难点分析：查看系统识别的重点和难点内容
   - 讲解脚本：查看系统生成的课堂讲解脚本
   - 知识点可视化：查看知识点的可视化展示

### 3.3 作业批改辅助

1. **进入页面**：点击导航菜单中的「作业批改」
2. **填写信息**：
   - 选择学科
   - 选择年级
   - 输入作业主题
   - 输入作业内容或题目
   - 输入学生答案
3. **功能操作**：
   - 生成批改标准：点击「生成批改标准」按钮，获取作业批改的参考标准
   - 生成反馈建议：点击「生成反馈建议」按钮，获取针对学生答案的个性化反馈
   - 分析学习情况：点击「分析学习情况」按钮，获取学生学习情况的分析报告

### 3.4 资源管理

1. **进入页面**：点击导航菜单中的「资源管理」
2. **资源操作**：
   - 上传资源：上传本地教学资源
   - 分类管理：对资源进行分类整理
   - 搜索资源：通过关键词搜索资源
   - 下载资源：下载已上传的资源

### 3.5 教师培训

1. **进入页面**：点击导航菜单中的「教师培训」
2. **培训内容**：
   - 教学方法：学习先进的教学方法和技巧
   - 课程标准：了解最新的课程标准和要求
   - 专业发展：获取教师专业发展的相关资源

## 4. 常见问题与解决方案

### 4.1 页面加载缓慢

**解决方案**：
- 检查网络连接是否稳定
- 清除浏览器缓存
- 尝试使用其他浏览器

### 4.2 生成内容不符合预期

**解决方案**：
- 提供更详细的教学信息
- 尝试调整输入参数
- 选择不同的模板（针对课件生成）

### 4.3 无法导出课件

**解决方案**：
- 检查浏览器是否允许弹出窗口和下载
- 确保浏览器版本是最新的
- 尝试使用不同的浏览器

### 4.4 系统功能无响应

**解决方案**：
- 刷新页面
- 检查网络连接
- 清除浏览器缓存后重新访问
