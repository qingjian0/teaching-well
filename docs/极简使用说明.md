# 乡村教师教学助手极简使用说明

## 系统访问

1. **本地访问**：直接打开项目目录中的 `index.html` 文件
2. **服务器访问**：通过服务器地址访问（如 http://example.com/teaching-well）

## 核心功能使用

### 1. 课件生成
- 进入「课件生成」页面
- 选择学科、年级，输入教学主题
- 选择课件模板
- 点击「生成课件」按钮
- 预览或导出生成的课件

### 2. 重难点解析
- 进入「重难点解析」页面
- 输入教学内容
- 点击「分析重难点」按钮
- 查看分析结果和讲解脚本

### 3. 作业批改辅助
- 进入「作业批改」页面
- 输入作业信息和学生答案
- 点击「生成批改标准」或「生成反馈建议」
- 查看批改结果和分析报告

### 4. 资源管理
- 进入「资源管理」页面
- 上传、分类、搜索和下载教学资源

### 5. 教师培训
- 进入「教师培训」页面
- 浏览和学习教学方法、课程标准等资源

## 系统要求
- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- 网络连接（首次加载需要联网）

## 常见问题
- **页面加载缓慢**：清除浏览器缓存，检查网络连接
- **生成内容不符合预期**：提供更详细的教学信息，调整输入参数
- **功能无响应**：刷新页面，检查网络连接
